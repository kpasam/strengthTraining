# Change Log — March 26, 2026

---

## Bug Fixes & Hardening

### Loading hang fixed
- Added a 10-second `AbortController` timeout to `fetchPlan` so the loading state can never get stuck indefinitely.
- Added `fetchError` state with a visible Retry button when the fetch fails or times out.

### Stale service worker cache fixed
- `sw.js`: stopped pre-caching HTML routes — auth-gated pages were being cached without credentials, causing the login-page HTML to be served at `/` on repeat visits.
- Excluded `/_next/` JS chunks from the cache — Turbopack reuses filenames across rebuilds, so stale JS was running after hot reloads.
- Bumped cache name to `v3` to force eviction of all old cached entries.

### Broken PWA icons fixed
- `icon-192.png` and `icon-512.png` were 1×1 px placeholders. Replaced with correct-sized icons.
- Added missing `favicon.ico`.

### Error boundary added
- New `ErrorBoundary` component catches uncaught render crashes and shows a recoverable error screen instead of a blank page.
- Wrapped app `children` in `ErrorBoundary` inside `layout.tsx`.

### Rest Day UI
- Home page now shows a proper "Rest Day" card (with 🛋️) when no workout plan exists for the selected date, instead of an empty or broken state.

### README
- Added missing `db:push` step required on first run.

## UI Improvements

### Settings modal (gear icon)
- Added a `SettingsModal` bottom-sheet component accessible via a gear icon (⚙) in the header.
- Moved the following out of the header into Settings:
  - **Sync from Google Slides** (was a top-bar button)
  - **Export Workout Data (CSV)** (was a top-bar link)
  - **Theme toggle** (Light / Dark)
  - **Log Out**
- Header now shows only the date navigator and the gear icon — significantly less cluttered.

### Date navigation arrows
- Replaced the raw `<input type="date">` in the header with **‹ / ›** prev/next day buttons.
- The date label between the arrows remains tappable to open the native date picker for jumping to arbitrary dates.

### "Mark complete early" demoted
- The prominent full-width red "COMPLETE WORKOUT (FINISH EARLY)" button is now a small, low-contrast secondary text link (`mark complete early`) below the Start button — reduces visual noise for the common case.

### Rest day hint copy updated
- Changed "hit Sync to re-pull from Google Slides" → "open Settings to sync from Google Slides" to match the new UI location.
